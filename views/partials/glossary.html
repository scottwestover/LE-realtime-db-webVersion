Name Description
Agent ID The id of the agent in the response.
Total The total time the agent was in this state (seconds).
Chatting The total time the agent was chatting in this state (seconds).
Not Chatting The total time the agent was not chatting in this state (seconds).
Chatting At Max Concurrency The amount of time the agent was chatting concurrently with the maximum allowed chats in this state (seconds).
Name Description
Skill ID The id of the skill in the response.
Total Time To Answer The total time, in seconds, that the visitors spent in the queue before being connected to an agent.
Entered Queue The number of visitors that entered the queue.
Average Time To Answer The average time, in seconds, that a visitor spent in the queue before being connected to an agent. Calculation: totalTimeToAnswer / connectedEng.
Abandonment Rate The number of visitors that abandoned the queue out of the total visitors that started an engagement. Calculation: abandonedEng / (abandonedEng + connectedEng).
Abandoned Engagements The number of visitors that abandoned the queue before being connected to an agent.
Connected Engagements The number of visitors that were connected to an agent.
Total Time To Abandon Total time visitors spend in queue before abandon.
Average Time To Abandon The average time visitors spend in queue before abandon. Calculation: totalTimeToAbandon / abandonedEng.
Max Queue Size The maximum queue size within the given time frame.
Min Queue Size The minimum queue size within the given time frame.
Average Queue Size The average queue size within the given time frame (rounded). Calculated as queueSizeSum / queueSizeCount.
Current Queue Size The current queue size.
Queue Size Sum Sum of all queue sizes received.
Queue Size Count Total number of queue size measures made.
Current Available Slots The current available slots.
Max Available Slots The maximum available slots within the given time frame.
Min Available Slots The minimum available slots within the given time frame.
Average Available Slots The average available slots within the given time frame (rounded). Calculated as availableSlotsSum / availableSlotsCount.
Available Sum Sum of all available slots received.
Available Count Total number of available slots measures made.
Name Description
Skill ID The id of the skill in the response.
Agent ID The id of the agent in the response.
Total Interactive Chats The number of interactive chats that took place in the current interval/time frame.
Total Non-Interactive Chats The number of non-interactive chats that took place in the current interval/time frame.
Total Handling Time The total time in seconds that was spent handing the interactive chats in the current interval/time frame.
Total Non-Interactive Handling Time The total time in seconds that was spent handing the non-interactive chats in the current interval/time frame.
Connected Engagements The number of chats connected to an agent in the current interval/time frame.
Name Description
Skill ID The id of the skill in the response.
Current Avaialble Slots The current queue number of available slots.
Current Queue Size The current queue size.
Name Description
Seconds The start of the SLA bucket in seconds.
Accumulated The percentage of chats that fall in this bucket or in earlier buckets (buckets for shorter queue wait periods).
Percentage From Total The percentage of chats that fall in this bucket.
Chats The number of chats with wait time that falls in this bucket.
